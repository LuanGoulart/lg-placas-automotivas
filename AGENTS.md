# LG Placas Automotivas - Guia para Agentes de IA

## Visão Geral do Projeto

Este é um site estático para uma empresa de estampagem de placas automotivas localizada em Itajubá, Minas Gerais. O site oferece serviços de:

- Placas Mercosul para carros e motos
- Segunda via de placas (perda/furto)
- Primeiro emplacamento
- Atendimento regional em Minas Gerais

## Estrutura do Site

### Páginas Principais

- `index.html` - Página inicial com serviços, depoimentos e FAQ
- `produtos.html` - Catálogo de produtos
- `placa-carro-itajuba.html` - Serviços específicos para carros
- `placa-moto-itajuba.html` - Serviços específicos para motos
- `placas-mercosul-itajuba-regiao.html` - Placas Mercosul regionais
- `segunda-via-placa-itajuba.html` - Segunda via de placas

### Assets

- `assets/css/styles.css` - Estilos organizados por seções
- `assets/js/` - Scripts JavaScript
- `imagens/` - Imagens do site

## Tecnologias Utilizadas

- **HTML5** semântico
- **Tailwind CSS** (CDN) para utilitários
- **CSS customizado** em `styles.css`
- **JavaScript vanilla** para interatividade
- **Schema.org markup** para SEO
- **GitHub Pages** para hospedagem

## Convenções de Desenvolvimento

### CSS

- Estilos organizados em seções comentadas
- Uso de classes Tailwind para layout
- CSS customizado para componentes específicos
- Responsividade mobile-first

### JavaScript

- Código vanilla (sem frameworks)
- Funções modulares
- Event listeners para interatividade
- Integração com WhatsApp

### SEO

- Meta tags otimizadas com palavra-chave "emplacamento"

- Sitemap e robots.txt configurados
- Open Graph para compartilhamento
